Calcular subredes online

Esta herramienta permite calcular subredes online introduciendo una IP y un prefijo CIDR para obtener red, broadcast, máscara y rango de hosts.

¿Para qué sirve?

Sirve para administración de redes, diseño de direccionamiento, configuración de equipos, soporte técnico y tareas de sistemas.

Qué resultados ofrece

Obtienes red, broadcast, máscara decimal, wildcard, primer host, último host y número de hosts válidos.

Ventajas de usar esta herramienta

Permite comprobar cálculos rápidamente sin usar comandos ni hojas de cálculo, directamente desde el navegador.

Preguntas frecuentes

Respuestas rápidas sobre esta herramienta.

¿Qué calcula esta herramienta?

Calcula la red, broadcast, máscara, wildcard, rango de hosts y número de hosts válidos a partir de una IP y un prefijo CIDR.

¿Sirve para IPv4?

Sí. Esta versión está orientada a direcciones IPv4.

¿Puedo introducir cualquier prefijo CIDR?

Sí, desde /0 hasta /32.

¿Se envían mis datos a un servidor?

No. Todo se calcula localmente en tu navegador.
