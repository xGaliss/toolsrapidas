Qué hace esta herramienta

Genera un bloque JSON-LD `BreadcrumbList` con `ListItem`, `position`, `name` e `item` para que puedas marcar rutas de navegación de forma limpia y rápida.

Cuándo merece la pena usar schema breadcrumb

Tiene sentido cuando una página forma parte de una estructura jerárquica clara y el usuario puede entender dónde está dentro del sitio.

Qué no conviene hacer

No uses breadcrumbs artificiales ni niveles que no existan en la navegación real. El marcado debe reflejar lo que ve el usuario, no inventar estructura solo para SEO.

Preguntas frecuentes

Dudas habituales sobre el generador de schema breadcrumb.

¿Qué tipo de schema genera?

Genera JSON-LD con `BreadcrumbList` y elementos `ListItem` listos para copiar en tu página.

¿Necesito mostrar breadcrumbs también en la página?

Es lo recomendable. El marcado debería corresponderse con una navegación visible o al menos coherente con la estructura real del sitio.

¿Sirve para páginas de herramientas?

Sí. En proyectos como ToolsRapidas ayuda a conectar cada tool con su categoría y con la home.

¿Puedo usar rutas relativas?

Sí. La herramienta puede convertirlas a URL completas usando el dominio base que introduzcas.
