Generador de SQL INSERT online

Crea sentencias INSERT SQL a partir de columnas y filas sencillas. Es útil para testing, migraciones simples, ejemplos y carga rápida de datos de prueba.

Qué necesitas

Solo necesitas indicar el nombre de la tabla, las columnas y una o varias filas con valores separados por comas.

Preguntas frecuentes

Respuestas rápidas sobre esta herramienta.

¿Sirve para bases de datos SQL comunes?

Sí. Genera una sintaxis base tipo INSERT INTO útil para muchos motores SQL.

¿Detecta números?

Sí. Los valores numéricos simples se dejan sin comillas.

¿Puedo generar varias filas?

Sí. Añade una fila por línea.
