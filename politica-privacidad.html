Política de privacidad

Datos que se recogen

ToolsRapidas no recoge datos personales directamente a través de formularios. Sin embargo, pueden recogerse datos anónimos de navegación mediante herramientas de análisis como Google Analytics.

Finalidad del tratamiento

Los datos recogidos se utilizan exclusivamente para analizar el uso del sitio web, mejorar la experiencia del usuario y optimizar el rendimiento de las herramientas.

Legitimación

La base legal para el tratamiento de los datos es el consentimiento del usuario, otorgado mediante la aceptación de cookies en el sitio web.

Conservación de los datos

Los datos se conservarán durante el tiempo necesario para cumplir con la finalidad para la que fueron recogidos o hasta que el usuario solicite su eliminación.

Derechos del usuario

Puedes ejercer tus derechos de acceso, rectificación, supresión o limitación del tratamiento enviando un correo electrónico a contacto@toolsrapidas.com.

Servicios de terceros

Este sitio web puede utilizar servicios de terceros como Google Analytics, que recopilan información de forma anónima sobre el uso del sitio web.

Seguridad

Se aplican medidas técnicas y organizativas adecuadas para garantizar la seguridad de los datos y evitar accesos no autorizados.

Privacidad en ToolsRapidas

ToolsRapidas prioriza la privacidad del usuario y minimiza la recopilación de datos, utilizando únicamente información necesaria para mejorar el funcionamiento del sitio.
